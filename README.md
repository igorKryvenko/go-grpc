# Demo gRPC
This is a simple demo project written on *golang* which demonstrate working with gRpc.

How to run
----------
Generate certificates for your hostname and put them into ***cert*** directory inside ***project*** directory <br>
 
 
Run Makefile
 ```
$ make
```
  
Start server
```
cd bin/
./server
```

Start client
```
cd bin/
./client
```
